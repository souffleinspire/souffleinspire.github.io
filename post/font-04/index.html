字体第四课：避坑指南-聊聊字体版权

· 2020-02-17 · # 字体 # 设计

前言

2019年，网传一公司实习生因侵权使用"微软雅黑"字体，全公司裁员42人，损失2860万，被戏称"一字千金"。虽然后来版权方出来辟谣，但仍在全网掀起了字体版权热议潮，曾经的"暴雪案"、"飘柔案"等也被重新拿出来讨论。 对设计师来说，字体版权绝对是说不出的痛。我的一位前同事因侵权使用，曾被索赔10万。去年，一个运营同学改动了 banner 上的字体，公司就收到了警告，索性没有造成经济损失。关于字体版权，还有多少坑？又该如何避开？

什么是字体版权？

为了说清楚字体版权，我特意去"中华人民共和国司法部"的网站查询了相关资料。严格来说，"字体版权"尚无明确严谨的司法领域概念。从语义上理解，"版权"是指文字、艺术、科学作品的作者对其作者享有的权利。"字体"的定义，在之前的文章里提到过其争议，中国法律中也没有明确解释。 关于字体版权是适用《专利法》还是《著作权法》也是分歧点。我综合查阅了两者法律条款、字体研发流程以及字体公开授权说明的信息，发现《著作权法》在版权保护上更全面、明确，所以应是重点参考的法律（但不意味着《专利法》不适用）。不管怎样，字体是明确受法律保护的，要使用就必须经得著作权人的同意。 字体为什么需要版权保护呢？一句话：现状就是十几亿国人在用汉字，但做字的却只有几百人。其实造字如同制药，都有一定的公益性。法律授权保护原创药品知识产权，才能鼓励药企研发新药，同理，对字体的版权保护到位了，就会有越来越多字体设计师投身其中。

字体授权类型

字体授权方式主要有以下几种:

  • 商业字体（Commercial）：用于商业发布（商业发布基本可以等同为公开发布，比如我写文章也算），应通过正规购买方式获得。
  • 免费字体（Freeware）：用户可以无限制（包括：个人使用、商业使用）免费使用字体，也叫做免费可商用字体。
  • 个人免费字体：个人用户可以无限制免费使用字体的版本，但不能用户商业发布。
  • 共享字体（Shareware）：包括试用字体（Trailware），根据共享字体作者的授权，用户可以从各种渠道免费下载和试用字体，认为满意后再向作者付费，简单说就是免费试用，付费商用。
  • 自由字体（Free Software）：使用者可以以任何目的使用字体，也可以无需字体开发方同意修改字体并可共享传播，甚至有时会提供字体源代码，但不能简单修改后申请版权。

当然，字体公司还可以根据使用范围等进行授权细分，方正字库就将授权分成 A、B、C、D 四类，每一类都有详尽的说明，对应的价格也很明确。字体授权的详细情况每个公司都不一样，但这些信息基本都可以在他们的官网上找到。

字体版权的期限

根据《著作权法》第二十一条，对公民的作品的法律保护期为作者终生及其死亡后五十年，如果是合作作品，截止于最后死亡作者的死后第五十年的12月31日。像黑体、宋体、楷体等，就是因已超过法定著作权保护期而成为开源字体。

那些号称免费商用的字体能放心使用吗？

当然不能。虽然我们在网络上可以很方便的找到不少开源字体，但这里其实有很多的坑。

  1. 很多下载资源本身就是盗版 搜索"字体下载"，可以得到大量资源，但是很多网站本身就是四处收集盗版字库的平台，所以你下载的也很可能是盗版。
  2. 字体版权是动态变化的 字库和手机 app 一样，也是可以更新迭代的，设计师千辛万苦打磨、完善的字体，可能就从原来的免费变成了收费，因此一些文章中推荐的免费商用字体也要注意准确性和时效性。
  3. 中文简体与繁体的版权区分 中国大陆和中国台湾在繁体字的基础字形规范方面多有不同，繁体字一般更为复杂，因此即使是同一字体，简体和繁体的版权也可能不同。
  4. 使用平台、使用场景区分 苹方是苹果公司研发的开源中文字体，但仅限于在苹果系统上使用。相应的，也有一些字体只能在安卓系统中使用。

侵权的后果

侵权最直接的后果当然是赔钱，侵权赔偿数额需要综合考虑字体的商业用途、使用数量、使用范围等因素，目前，法院对这类侵权的判决一般为一个字3000-10000元。如果字体设计在产品营利方面起到关键作用，赔偿金额还会更多。

如何使用字体而不侵权？

针对字体版权侵权问题，我们可以有3种措施来规避：

  1. 使用开源或已授权可以免费商业使用的字体 再次提醒，一定要确认版权后才能在商用场景下使用这些字体，最直接有效的方式就是去字体官网上查询。此外，也有相对简单一点的查询途径，比如：360查字体和字由。前者偏保守、不全面，主要作用是能检测电脑已安装字体的版权，建议查到没有免费商业版权的就可以停止使用了。后者全面，不过为保险起见，对于所谓的"免费可商用"字体也需要经过验证。
  2. 购买正版字体或单字做商业性使用 不要心存侥幸，现在靠投诉字体侵权捞金的人很多，拿到商业授权书且在规定范围内使用才能避免接到律师函。顺便一提，为减少不必要的打扰，有正版授权的话，可以适当通过一些途径"宣誓主权"。
  3. 自创或者在他人字体基础上进行修改再创作（达到30%的区别度） 在原本就是开源字体基础上修改创造，基本不用担心侵权；在有商业版权保护的字体基础上修改，必须达到30%以上的区分度才可以；自创字体也需要像写论文一样查重，避免与版权字体区分度不明显。像下面这样简单的更改有版权字体万万不可！
