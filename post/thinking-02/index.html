我的野蛮生长

· 2019-07-07 · # 我 # 随笔

听说有读者想知道我是「如何做设计方面的训练的」，虽然简短地回答了一下，但是现在想来说的稍有不妥，所以就专门花一篇文章讲讲吧！

这是一个很难的问题

不得不说，细想这个问题，其实挺难回答。读者之中，若不是认识的人，我其实一点也不了解。不知道大家的职业，不知道大家学习设计的目的，不知道大家的期望达到的水平，那我不能说出任何指点性的意见。

我的成长经历，让我明白任何人说的话，都不一定绝对正确。每个人的观点都和自己的经验有关，观点就是观点，只有被反复验证才可能成为真理。如果始终保持怀疑的心，那就容易辨别出什么话该听，什么话可以置之不理。

以下要讲述的是我学习设计的经验，并不一定适合每个人，而且我个人也还有很长的路要走，请择之而行。

热爱使我义无反顾

对于一个学理工科的女生来说，设计好像一辈子都与我无瓜。哪知小时候因为画画、做手工而萌生出来的小种子早已埋藏在心中。直到大三，开始接触到 UI 设计，那粒种子得以迸发生机。

是的，我学的专业和设计没有半毛钱关系。在大家如火如荼准备考研或者找工作时，我开始了一年多的学习，从清晨，到迟暮，至深夜。说心里不打鼓是骗人的，毕竟我直到毕业，也没有找到工作，成了班上唯一一个没有着落的"钉子户"。

很多人不喜欢没有功底的设计师，我也不鼓励像我一样。但是如果你心中有这样一份热爱，也不宜轻易的放弃。如果当初没有义无反顾，也许我会抱憾终身。

那如果说您只是想浅尝辄止地学习一些，我觉得没有问题。听好多人说过好多次，「智能化 和 通才化是未来的方向」。

之所以能走下去......

现在是真正的主题时间。完全自学，凭什么能走到现在？我认为有几个选择或是认识是使我受益终身的：

1. 舍不得孩子，套不着狼 让你付出了很多的东西，你会印象更深刻。手里揣着1万的时候，花了6000多买了台 MacBook Air，花了3000多买课程，换来了更好的内容、更顺畅的体验和更充足的学习动力，当然了，还有我同学和老师异样的目光。不过，管它的，就当羡慕好了！

课程需要好好选，毕竟花了"大价钱"，性价比得高。我是先从软件学起的，不多，也就八九个吧😂 这部分课程400多元，当时在网易云课堂上找的，软件很容易上手，不值得花那么多钱。其余的2000多，是在半年之后，在网易云上参加另一个微专业课程。

2. 不一定非有基础，但不能不重视基础 那个微专业课全称「《高级 UI 设计师》微专业课程」，可想而知难度非一般。直奔高级除了时间紧之外，更重要的是我知道更缺乏的是理论基础，也就是没有系统的知识框架。这是我纯粹学网课的弊端，必须有所弥补。

虽然工作了，我也在想尽办法提升自己对设计基础的理解，看书、看教程，包括写文章一定程度上也是对自己的督促。如果不继续，基础不牢会成为我未来的阻碍，我在尽力避免这一点。

3. 和自己对话 学了这么多软件，现在工作也就用到两三个，所以对我来说主要的意义就是，那段时间让我触碰到互联网产品设计几乎全门类。我来数一数，大致包括：UI、交互、运营、插画、动效、品牌。

实习面试的时候，设计总监和我说"设计师要有自己的特长"，以前不以为意，后来确实让自己有了更清晰的认识。实习主要做动效，一开始做得慢，后来速度上去了，开始要求自己的质量。再后来找工作的时候，专门做了个求职动画作品，算是一个亮点。

因为学的是工科，在逻辑方面可能稍强一点，就一直想往交互设计师的方向走。不敢说做得有多好，但我觉得路是对的。

4. 向 人 学习 向人学习，这里的「人」是每个人，身边的每个人。真正优秀的人极少藏着掖着，他们乐于分享。虽说不能尽信，但「优秀」的理由可以借鉴。

我每天都能看到身边人值得学习的地方，某些方面，我在别人那儿就是"渣渣"。看到这个同事 icon 设计得漂亮，那个同事插画画得轻松，有的人逻辑感比我强，我都想让他们教。当然不能全学，也无法全学，但相信「三人行，必有我师」令我受益良多。

5. 把所想的变成所做的 我最佩服的是我的勇气，说真的！不怕大家笑话，当时那份实习的工作是我自己直接冲到公司去的。不是没有在网上投简历，而是迟迟没有收到回信。

我留在了那里，才有了更多的故事。在快要离开实习公司的时候，我开始写文章。写文章很费脑子啊啊啊～边写边梳理，好多知识就连贯起来了。后来和尝试朋友一起做产品，我负责设计，在那期间，也学了一点插画，锻炼了一些产品能力。然后，开始以"交互设计师"的身份工作。

这些都是我想做的，我真的就去做了。还有好多想做的，我正在做、计划做。当知道自己有任务在身的时候，劲头儿都很足，学东西就很快了。

关于设计，我常用的资源

资源、工具这东西，我是一直觉得得来全不费功夫的。常用到的资源比较少，也就是设计师常用那些。现在大致可以分享一下，目前了解得不太多，以后再慢慢补充。

灵感类

  • Behance：基本是全世界设计师一些比较完整的项目，质量较高，我用来找灵感和找 mockup（今年被墙了，自己想办法吧）。网址：https://www.behance.net/
  • Dribbble：找一些小插画、动效什么的很方便，自己的小作品也可以上传。网址：https://dribbble.com/
  • Lapa ninja：也是搜集优质网站的平台，对有网页设计需求的挺好。网址：https://www.lapa.ninja/
  • 站酷：国内设计分享平台。因为作品差异较大，我其实不怎么去，但是应该是做得越来越好了，上传自己的设计作品挺不错的。网址：https://www.zcool.com.cn/

教程类

  • 网易云课堂：普通课程一般，微专业课程可以按需学习，其他不知道，但设计方面的微专业课基本都是网易系的设计师出的。网址：https://study.163.com/
  • Design + Code：适合想要成为全栈设计师或者全栈开发者的朋友，有基础有难度，关键会推荐很多好资源。网址：https://designcode.io/
  • 虎课网：实用类和基础类的课程偏多，我就用来快速学技能的。网址：https://huke88.com/

公众号

  • 研习设：我目前唯一推荐的设计公众号，什么都将，而且每一篇都很用心。在微信里搜索 研习设 即可。
