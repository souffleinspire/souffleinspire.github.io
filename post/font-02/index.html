字体第二课：常用der英文字体

· 2020-01-06 · # 字体 # 设计

前言

在第一课，已经介绍了中西文字体基础，今天再进一步讲讲西文（英文）字体以及我在工作中如何使用它们。

西文字体的历史

很久很久以前，为了记录生活中的信息，人类开始用木炭等在岩石上写写画画，这些符号成为原始象形文字的雏形。 为了让更多人看懂内容，起到沟通和传播的作用，人们需要使用一致的语言体系，岩壁上的文字也慢慢标准化。 随后，易于携带的纸张被发明出来，但是制作成本高昂，用来书写复杂的文字很浪费。这一阶段，人们开始摸索如何简化文字。 而在古罗马，文字系统已经比较发达，但仍以雕刻形式存在。彼时小写字母还没发明出来，字母是全大写。Trajan、Stempel、Hercvlanum 等字体就是古罗马风格（Old roman），全大写的衬线体。 四世纪，文字主要书写在羊皮纸上，因为造价昂贵，所以当时的字体高度小，字距也小，代表字体是 Uncial（安色尔）体，到八世纪成为圣经的主要字体。从图片可以看出来，这一时期字体是大小写混用的。 卡洛林王朝时期兴起的 Carolingian（卡洛林）字体，外形上已接近现代英文字体，也开始大小写成对使用。查理曼时期，为保证抄本内容的准确性，Carolingian 小写字体成为中世纪的"官方字体"。 到了十二世纪，大学兴起，越来越多学生接受高等教育，对书本的需求量也增加了，市场很需要快速复制书籍的办法。十五世纪，古腾堡发明活字印刷，实现了标准、可重复的印刷生产。当时已开始文艺复兴，带有明显平头笔书写特征的 Blackletter （哥特黑体）、x 字高（字母 x 的高度，是用来度量字体基本高度的标准）更小的意大利斜体和复古的古罗马体是潮流字体。 随着纸媒的发展、人们创意的爆发以及新力量的崛起，越来越多的英文字体诞生。比如，十八世纪的洛可可（Rococo）风格，字体细腻柔美、曲线繁杂，有很明显的女性感。 二十世纪，互联网开始崭露头角，字体设计也迎来新的繁荣期。屏幕上用花哨的字体会有一定的虚化，所以无衬线体更加吃香，出现了 Helvetica 这样的经典字体。当然，线下场景的丰富性同样催生了更多艺术字体。

英文字体分类

除了第一课里的两大基本分类----衬线体和无衬线体，还有这两者之外的其他字体，但不做介绍了，我们来细化下衬线体、无衬线体。

按照风格分类，衬线体可分为旧体（old style）和现代体（modern style）。「旧体」带有手写风，笔尖有固定倾斜角度的书写痕迹，字母 O 的较细部分连线是斜线。「现代体」字母 O 的较细部分连线是垂直的，比例工整，没有手写痕迹。「旧体」并不过时，其中一些字体反倒比「现代体」更适合长文阅读。 按照衬线不同分类，还可分为三类：

  • 曲线角度的"支架衬线体"，字体风格亲切、传统，代表字体有：Times New Roman，Baskerville，Caslon，Georgia。
  • 连接处为细直线的"发丝衬线体"，现代感很强，适合大标题用，比如 Didot，Bodoni。
  • 连接处粗厚的"板状衬线体"，显得粗壮有力，也适合标题使用，有些怀旧氛围。

无衬线体也有一些分类，个人认为不需要掌握，看到不同字体时的感觉能指导你使用它们。

个人常用英文字体

下面是我个人常用的英文字体，但是不意味着任何情况下都可以尽情使用。强调一下，使用字体一定要注意版权信息，避免侵权使用遭受损失，所以即使是设计师，电脑上也不用装很多字体，反正不能用也白搭。关于字体版权的问题，我会写另外一篇文章。

衬线体

  • Times New Roman Times New Roman 是我大多数使用衬线体时的首选字体。它是现代-支架衬线体，最初为泰晤士报设计，清晰易辨认，感觉上属于中性的衬线体。
  • Didot 现代-发丝衬线体，粗细对比强烈，具有现代、时尚之美。敲重点，时尚、高档的字体。
  • Garamond Garamond 这套字体看着很古老，实际上也很古老，可以追溯到16世纪，但是它线条匀称流畅，可读性强而且优雅，需要一些复古感的时候会用这个字体。下图其实是 Adobe Garamond Pro，其斜体比 Garamond 要好看些。

非衬线体

  • Helvetica 经典中的经典字体，结构严谨，因为没有什么特点反倒是被广泛使用，不会错的字体。字重从细体到粗体，很全面。雀巢（Nestle），丰田（Toyota），宝马（BMW）等众多企业的商标都曾使用 Helvetica。
  • Robot 安卓系统默认英文字体，字型挺不错的（不过我不太喜欢它的 Q 字母），Robot 字体中数字偏窄，比较简洁。
  • Futura Futura 字体跟随者包豪斯设计浪潮流行开来，具有完全的几何感。Futura 的字母里所有弯曲都被去掉了，比如 t、j 的勾勾都没有，所以 j 很像 i，有一定的辨识难度，平常主要用在标题、装饰上。Futura 还有窄体，在字体设计中很少见，很喜欢这点。
  • Avenir 用 Futura 显得过于尖锐和硬朗的时候，Avenir 则是很好的替代，它的数字也很优秀，用来展示英文为主的界面效果非常好。
  • San Francisco OS 系统默认英文字体，比 Robot 更饱满一些，但从数字上看，不是很现代。

总结

通过了解英文字体的发展历史，让我们了解了英文的主要分类及其特征。"字体不在多，实用则灵"，不做字体设计，没必要下载各种花式字体。最后再次提醒，字体千万种，版权第一要重视。
